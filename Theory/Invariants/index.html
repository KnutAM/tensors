Tensor invariants

One important application for this is modeling of material behavior. For example, the von Mises effective stress, (2/3)σdev:σdev\sqrt{(2/3)\boldsymbol{ \sigma}^{ \mathrm{dev} }:\boldsymbol{ \sigma}^{ \mathrm{dev} }}(2/3)σdev:σdev​ is an invariant, because the double contraction with itself, and the trace (used to get the deviatoric part), is an invariant.

Eigenvalues

Another set of invariants for a 2nd order tensor are its eigenvalues, λi\lambda_iλi​. The eigenvalues and corresponding eigenvectors are defined by

av‾i=λiv‾i\begin{aligned} \boldsymbol{ a}\underline{\boldsymbol{ v}}_i = \lambda_i \underline{\boldsymbol{ v}}_i \end{aligned}av​i​=λi​v​i​​

The non-zero vectors v‾i\underline{\boldsymbol{ v}}_iv​i​ that fulfill this equation are called eigenvectors of a\boldsymbol{ a}a, and the corresponding values λi\lambda_iλi​ are called eigenvalues.

As for matrices, we can determine the eigenvalues by solving the characterstic equation

det(a−λI)=0\begin{aligned} \mathrm{det}\left(\boldsymbol{ a}-\lambda \boldsymbol{ I}\right) = 0 \end{aligned}det(a−λI)=0​

Given the eigenvalues, λi\lambda_iλi​, each corresponding eigenvector v‾i\underline{\boldsymbol{ v}}_iv​i​ can be obtained by solving

(a−λiI)v‾i=0‾\begin{aligned} \left(\boldsymbol{ a}-\lambda_i \boldsymbol{ I}\right)\underline{\boldsymbol{ v}}_i = \underline{\boldsymbol{ 0}} \end{aligned}(a−λi​I)v​i​=0​​

for a nonzero v‾i\underline{\boldsymbol{ v}}_iv​i​. As we have equality with zero, the magnitude of v‾i\underline{\boldsymbol{ v}}_iv​i​ is arbitrary. In practice, it is common to normalize the eigenvectors.

Spectral decomposition

Above, we introduced the right eigenvectors v‾\underline{\boldsymbol{ v}}v​, such that av‾=λv‾\boldsymbol{ a}\underline{\boldsymbol{ v}}=\lambda\underline{\boldsymbol{ v}}av​=λv​. It is also possible to introduce the left eigenvectors, w‾\underline{\boldsymbol{ w}}w​, such that w‾a=λw‾\underline{\boldsymbol{ w}}\boldsymbol{ a}=\lambda\underline{\boldsymbol{ w}}w​a=λw​. This leads to the equivalent characteristic equation, det(aT−λI)=det(a−λI)=0\mathrm{det}\left(\boldsymbol{ a}^{\mathrm{T}}-\lambda\boldsymbol{ I}\right) = \mathrm{det}\left(\boldsymbol{ a}-\lambda\boldsymbol{ I}\right) = 0det(aT−λI)=det(a−λI)=0. Hence, the eigenvalues are the same. However, the eigenvectors are not always the same. If a\boldsymbol{ a}a is symmetric, then they are equal.

Invariants tensors which are not 2nd order

Invariants are not only restricted to 2nd order tensors. For example, the length of a vector is an invariant. Actually, the norm of any tensor is an invariant.

Invariants of multiple tensors (combined invariants)

It is also possible to define combined invariants that depend on multiple tensors. These remain invariant as long as all involved tensors are transformed the same way. A common example is the angle between two tensors, a:b/(∣∣a∣∣∣∣b∣∣)\boldsymbol{ a}:\boldsymbol{ b}/(\left\vert\left\vert \boldsymbol{ a}\right\vert\right\vert\left\vert\left\vert \boldsymbol{ b}\right\vert\right\vert)a:b/(∣∣a∣∣∣∣b∣∣). From this expression, we can also easily see that the double contraction a:b\boldsymbol{ a}:\boldsymbol{ b}a:b is invariant.
