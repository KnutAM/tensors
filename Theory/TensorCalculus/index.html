Differentiating tensor expressions

On this page, we will work with expressions of the form

du‾dv‾=duidvje‾i⊗e‾j\begin{aligned} \frac{\mathrm{d} \underline{\boldsymbol{ u}}}{\mathrm{d} \underline{\boldsymbol{ v}}} = \frac{\mathrm{d} u_i}{\mathrm{d} v_j} \underline{\boldsymbol{ e}}_{ i}\otimes\underline{\boldsymbol{ e}}_{ j} \end{aligned}dv​du​​=dvj​dui​​e​i​⊗e​j​​

which is true for constant orthonormal coordinate systems. With constant, we mean that the base vectors are constant in space. Hence, we can always differentiate the coefficients. And since we can consider each free coefficient, e.g. u1u_1u1​, as a scalar value, we can apply basic calculus rules, for example the chain and product rules.

Differentiating tensor function wrt. scalar

If we consider a=f(x)=xb\boldsymbol{ a} = f(x) = x\boldsymbol{ b}a=f(x)=xb, then

dadx=dxbijdxe‾i⊗e‾j=bije‾i⊗e‾j=b\begin{aligned} \frac{\mathrm{d} \boldsymbol{ a}}{\mathrm{d} x} = \frac{\mathrm{d} x b_{ij}}{\mathrm{d} x} \underline{\boldsymbol{ e}}_{ i}\otimes\underline{\boldsymbol{ e}}_{ j} = b_{ij} \underline{\boldsymbol{ e}}_{ i}\otimes\underline{\boldsymbol{ e}}_{ j} = \boldsymbol{ b} \end{aligned}dxda​=dxdxbij​​e​i​⊗e​j​=bij​e​i​⊗e​j​=b​

because bijb_{ij}bij​ doesn't depend on xxx.

Differentiating tensor function wrt. tensor

Let's first consider the differentiating a tensor wrt. itself. For a first-order tensor, we have

du‾du‾=duiduje‾i⊗e‾jduiduj=δijdu‾du‾=I\begin{aligned} \frac{\mathrm{d} \underline{\boldsymbol{ u}}}{\mathrm{d} \underline{\boldsymbol{ u}}} &= \frac{\mathrm{d} u_i}{\mathrm{d} u_j} \underline{\boldsymbol{ e}}_{ i}\otimes\underline{\boldsymbol{ e}}_{ j}\\ \frac{\mathrm{d} u_i}{\mathrm{d} u_j} &= \delta_{ij} \\ \frac{\mathrm{d} \underline{\boldsymbol{ u}}}{\mathrm{d} \underline{\boldsymbol{ u}}} &= \boldsymbol{ I} \end{aligned}du​du​​duj​dui​​du​du​​​=duj​dui​​e​i​⊗e​j​=δij​=I​

We see that if i=ji=ji=j then dui/duj=1\mathrm{d} u_i/\mathrm{d} u_j = 1dui​/duj​=1. If i≠ji\neq ji=j, then dui/duj=0\mathrm{d} u_i/\mathrm{d} u_j = 0dui​/duj​=0

If we now consider a 2nd order tensor, we have

dada=daijdakle‾i⊗e‾j⊗e‾k⊗e‾ldaijdakl=δikδjldada= I\begin{aligned} \frac{\mathrm{d} \boldsymbol{ a}}{\mathrm{d} \boldsymbol{ a}} &= \frac{\mathrm{d} a_{ij}}{\mathrm{d} a_{kl}} \underline{\boldsymbol{ e}}_{ i}\otimes\underline{\boldsymbol{ e}}_{ j}\otimes\underline{\boldsymbol{ e}}_{ k}\otimes\underline{\boldsymbol{ e}}_{ l} \\ \frac{\mathrm{d} a_{ij}}{\mathrm{d} a_{kl}} &= \delta_{ik}\delta_{jl}\\ \frac{\mathrm{d} \boldsymbol{ a}}{\mathrm{d} \boldsymbol{ a}} &= \textbf{\textsf{ I}} \end{aligned}dada​dakl​daij​​dada​​=dakl​daij​​e​i​⊗e​j​⊗e​k​⊗e​l​=δik​δjl​= I​

We can see that if i=ki=ki=k and j=lj=lj=l, then daij/dakl=1\mathrm{d} a_{ij}/\mathrm{d} a_{kl} = 1daij​/dakl​=1. Otherwise, it is zero. In other words: δikδjl\delta_{ik}\delta_{jl}δik​δjl​.

To consider a more complicated example, we look at

d[v‾a]dv‾=dvkakidvje‾i⊗e‾jdvkakidvj=dvkdvjaki=δkjaki=ajid[v‾a]dv‾=aT\begin{aligned} \frac{\mathrm{d} \left[\underline{\boldsymbol{ v}}\boldsymbol{ a}\right]}{\mathrm{d} \underline{\boldsymbol{ v}}} &= \frac{\mathrm{d} v_k a_{ki}}{\mathrm{d} v_j} \underline{\boldsymbol{ e}}_{ i}\otimes\underline{\boldsymbol{ e}}_{ j} \\ \frac{\mathrm{d} v_k a_{ki}}{\mathrm{d} v_j} &= \frac{\mathrm{d} v_k}{\mathrm{d} v_j} a_{ki} = \delta_{kj} a_{ki} = a_{ji} \\ \frac{\mathrm{d} \left[\underline{\boldsymbol{ v}}\boldsymbol{ a}\right]}{\mathrm{d} \underline{\boldsymbol{ v}}} &= \boldsymbol{ a}^{\mathrm{T}} \end{aligned}dv​d[v​a]​dvj​dvk​aki​​dv​d[v​a]​​=dvj​dvk​aki​​e​i​⊗e​j​=dvj​dvk​​aki​=δkj​aki​=aji​=aT​

Differentiating scalar function wrt. tensor

If we consider y=f(a)=a:ay = f(\boldsymbol{ a}) = \boldsymbol{ a}:\boldsymbol{ a}y=f(a)=a:a, then

dyda=daklakldaije‾i⊗e‾j=[dakldaijakl+akldakldaij]e‾i⊗e‾j=[δkiδljakl+aklδkiδlj]e‾i⊗e‾j=[aij+aij]e‾i⊗e‾j=2aije‾i⊗e‾j=2a\begin{aligned} \frac{\mathrm{d} y}{\mathrm{d} \boldsymbol{ a}} &= \frac{\mathrm{d} a_{kl}a_{kl}}{\mathrm{d} a_{ij}} \underline{\boldsymbol{ e}}_{ i}\otimes\underline{\boldsymbol{ e}}_{ j}\\ &= \left[\frac{\mathrm{d} a_{kl}}{\mathrm{d} a_{ij}} a_{kl} + a_{kl} \frac{\mathrm{d} a_{kl}}{\mathrm{d} a_{ij}}\right]\underline{\boldsymbol{ e}}_{ i}\otimes\underline{\boldsymbol{ e}}_{ j} \\ &= \left[\delta_{ki}\delta_{lj} a_{kl} + a_{kl} \delta_{ki}\delta_{lj}\right]\underline{\boldsymbol{ e}}_{ i}\otimes\underline{\boldsymbol{ e}}_{ j} \\ &= \left[a_{ij} + a_{ij}\right]\underline{\boldsymbol{ e}}_{ i}\otimes\underline{\boldsymbol{ e}}_{ j} = 2a_{ij}\underline{\boldsymbol{ e}}_{ i}\otimes\underline{\boldsymbol{ e}}_{ j} = 2\boldsymbol{ a} \end{aligned}dady​​=daij​dakl​akl​​e​i​⊗e​j​=[daij​dakl​​akl​+akl​daij​dakl​​]e​i​⊗e​j​=[δki​δlj​akl​+akl​δki​δlj​]e​i​⊗e​j​=[aij​+aij​]e​i​⊗e​j​=2aij​e​i​⊗e​j​=2a​

Green-Gauss

The Green-Gauss theorem is often stated as

∫Vϕ[q‾⋅∇‾]dV=∫Sϕq‾⋅n‾dS−∫V[∇‾ϕ]⋅q‾dV\begin{aligned} \int_V \phi \left[\underline{\boldsymbol{ q}}\cdot\underline{\boldsymbol{ \nabla}}\right] \mathrm{d} V &= \int_S \phi \underline{\boldsymbol{ q}}\cdot \underline{\boldsymbol{ n}} \mathrm{d} S - \int_V \left[\underline{\boldsymbol{ \nabla}}\phi\right]\cdot \underline{\boldsymbol{ q}} \mathrm{d} V \end{aligned}∫V​ϕ[q​⋅∇​]dV​=∫S​ϕq​⋅n​dS−∫V​[∇​ϕ]⋅q​dV​

where VVV is a volume and SSS its bounding surface. n‾\underline{\boldsymbol{ n}}n​ is the outward pointing normal on SSS. ϕ\phiϕ is a scalar and q‾\underline{\boldsymbol{ q}}q​ a vector, both functions of the position x‾\underline{\boldsymbol{ x}}x​. If we write the corresponding expression in index notation, we have

∫Vϕ∂qi∂xidV=∫SϕqinidS−∫V∂ϕ∂xiqidV\begin{aligned} \int_V \phi \frac{\partial q_i}{\partial x_i} \mathrm{d} V &= \int_S \phi q_i n_i \mathrm{d} S - \int_V \frac{\partial \phi}{\partial x_i} q_i \mathrm{d} V\\ \end{aligned}∫V​ϕ∂xi​∂qi​​dV​=∫S​ϕqi​ni​dS−∫V​∂xi​∂ϕ​qi​dV​

Let's set up this equation for a vector instead of the scalar ϕ\phiϕ. By using the index notation, each free index is actually just a scalar so we effectively just write three equations of the same as above, changing ϕ\phiϕ to uku_kuk​ with u‾=uke‾k\underline{\boldsymbol{ u}}=u_k\underline{\boldsymbol{ e}}_{ k}u​=uk​e​k​.

∫Vuk∂qi∂xidV=∫SukqinidS−∫V∂uk∂xiqidV∫Vu‾⊗[q‾⋅∇‾]dV=∫Su‾⊗q‾⋅n‾dS−∫V[u‾⊗∇‾]⋅q‾dV\begin{aligned} \int_V u_k \frac{\partial q_i}{\partial x_i} \mathrm{d} V &= \int_S u_k q_i n_i \mathrm{d} S - \int_V \frac{\partial u_k}{\partial x_i} q_i \mathrm{d} V\\ \int_V \underline{\boldsymbol{ u}} \otimes \left[\underline{\boldsymbol{ q}}\cdot \underline{\boldsymbol{ \nabla}}\right] \mathrm{d} V &= \int_S \underline{\boldsymbol{ u}} \otimes \underline{\boldsymbol{ q}}\cdot\underline{\boldsymbol{ n}} \mathrm{d} S - \int_V \left[\underline{\boldsymbol{ u}} \otimes \underline{\boldsymbol{ \nabla}}\right]\cdot \underline{\boldsymbol{ q}} \mathrm{d} V \end{aligned}∫V​uk​∂xi​∂qi​​dV∫V​u​⊗[q​⋅∇​]dV​=∫S​uk​qi​ni​dS−∫V​∂xi​∂uk​​qi​dV=∫S​u​⊗q​⋅n​dS−∫V​[u​⊗∇​]⋅q​dV​

Similarly, we can write it exchanging the vector q‾\underline{\boldsymbol{ q}}q​ with a 2nd order tensor, a=aije‾i⊗e‾j\boldsymbol{ a}=a_{ij}\underline{\boldsymbol{ e}}_{ i}\otimes\underline{\boldsymbol{ e}}_{ j}a=aij​e​i​⊗e​j​

∫Vuk∂aij∂xidV=∫SukaijnidS−∫V∂uk∂xiaijdV∫Vu‾⊗[∇‾⋅a]dV=∫Su‾⊗n‾⋅adS−∫V[u‾⊗∇‾]⋅adV\begin{aligned} \int_V u_k \frac{\partial a_{ij}}{\partial x_i} \mathrm{d} V &= \int_S u_k a_{ij} n_i \mathrm{d} S - \int_V \frac{\partial u_k}{\partial x_i} a_{ij} \mathrm{d} V\\ \int_V \underline{\boldsymbol{ u}} \otimes \left[\underline{\boldsymbol{ \nabla}}\cdot \boldsymbol{ a} \right] \mathrm{d} V &= \int_S \underline{\boldsymbol{ u}} \otimes \underline{\boldsymbol{ n}}\cdot\boldsymbol{ a} \mathrm{d} S - \int_V \left[\underline{\boldsymbol{ u}} \otimes \underline{\boldsymbol{ \nabla}}\right]\cdot \boldsymbol{ a} \mathrm{d} V \end{aligned}∫V​uk​∂xi​∂aij​​dV∫V​u​⊗[∇​⋅a]dV​=∫S​uk​aij​ni​dS−∫V​∂xi​∂uk​​aij​dV=∫S​u​⊗n​⋅adS−∫V​[u​⊗∇​]⋅adV​

This expression represents 9 equations (kkk and jjj). If we sum the equations where k=jk=jk=j, i.e. multiply by δkj\delta_{kj}δkj​, we get

∫Vuj∂aij∂xidV=∫SujaijnidS−∫V∂uj∂xiaijdV∫Vu‾⋅[∇‾⋅a]dV=∫Sn‾⋅a⋅u‾dS−∫V[u‾⊗∇‾]:aTdV\begin{aligned} \int_V u_j \frac{\partial a_{ij}}{\partial x_i} \mathrm{d} V &= \int_S u_j a_{ij} n_i \mathrm{d} S - \int_V \frac{\partial u_j}{\partial x_i} a_{ij} \mathrm{d} V\\ \int_V \underline{\boldsymbol{ u}} \cdot \left[\underline{\boldsymbol{ \nabla}}\cdot \boldsymbol{ a} \right] \mathrm{d} V &= \int_S \underline{\boldsymbol{ n}}\cdot\boldsymbol{ a}\cdot\underline{\boldsymbol{ u}} \mathrm{d} S - \int_V \left[\underline{\boldsymbol{ u}} \otimes \underline{\boldsymbol{ \nabla}}\right] : \boldsymbol{ a}^{\mathrm{T}} \mathrm{d} V \end{aligned}∫V​uj​∂xi​∂aij​​dV∫V​u​⋅[∇​⋅a]dV​=∫S​uj​aij​ni​dS−∫V​∂xi​∂uj​​aij​dV=∫S​n​⋅a⋅u​dS−∫V​[u​⊗∇​]:aTdV​
