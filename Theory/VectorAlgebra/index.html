Vector algebra

In physics, vectors describe quantities that have a magnitude and a spatial direction, such as a position, a velocity, and a temperature gradient. We will later see that these vectors are, in fact, basically first order tensors. But for now, let's cover the basic rules of vector algebra as an introduction.

Consider the vectors x‾\underline{\boldsymbol{ x}}x​, u‾\underline{\boldsymbol{ u}}u​, v‾\underline{\boldsymbol{ v}}v​, and w‾\underline{\boldsymbol{ w}}w​ in the figure below.

The figure illustrates a the two first important properties of vector addition:

  • Addition is associative, i.e. u‾+v‾=v‾+u‾\underline{\boldsymbol{ u}}+\underline{\boldsymbol{ v}}=\underline{\boldsymbol{ v}}+\underline{\boldsymbol{ u}}u​+v​=v​+u​

  • Addition is commutative, i.e. (u‾+v‾)+w‾=u‾+(v‾+w‾)(\underline{\boldsymbol{ u}}+\underline{\boldsymbol{ v}}) + \underline{\boldsymbol{ w}} = \underline{\boldsymbol{ u}} + (\underline{\boldsymbol{ v}} + \underline{\boldsymbol{ w}})(u​+v​)+w​=u​+(v​+w​)

More formally, we can introduce a vector space V\mathbb{ V}V. This can for example be the real (3-dimensional) world, R3\mathbb{ R}^3R3. We can then define the full set of properties required for the vectors in V\mathbb{ V}V to be just vectors:

Vector summation

  • Associative addition (see above)

  • Commutative addition (see above)

  • There exists a unique zero vector, 0‾\underline{\boldsymbol{ 0}}0​, such that 0‾+v‾=v‾, ∀v‾∈V\underline{\boldsymbol{ 0}}+\underline{\boldsymbol{ v}}=\underline{\boldsymbol{ v}},\; \forall \underline{\boldsymbol{ v}}\in\mathbb{ V}0​+v​=v​,∀v​∈V

  • Commutative multiplication by scalar, av‾=v‾⋅aa\underline{\boldsymbol{ v}}=\underline{\boldsymbol{ v}}\cdot aav​=v​⋅a

Linear dependence

Assume that you have two different vectors in 2d and you want to construct a 3rd vector by scaling your vectors and adding them together. Since there are 2 dimensions, and you have two vectors, this should be possible, or?

However, what if your two vectors are pointing in the same direction (but have different length)? Then, you can only construct another vector in that direction. So even if you have two vectors, you are only spanning a line in your 2-dimensional space. I.e. you are only spanning a 1-dimensional space with your vectors. The reason is that these vectors are linearly dependent. Generally, we can formulate the definition of linear dependence as

Definition:
With i=1,2,⋯ ,Ni=1,2,\cdots,Ni=1,2,⋯,N, the vectors v‾i∈V\underline{\boldsymbol{ v}}_i\in\mathbb{ V}v​i​∈V are linearly dependent if there exists scalars ai∈Ra_i\in\mathbb{ R}ai​∈R such that aiv‾i=0‾a_i \underline{\boldsymbol{ v}}_i = \underline{\boldsymbol{ 0}}ai​v​i​=0​. If no such scalars exists, the vectors v‾i\underline{\boldsymbol{ v}}_iv​i​ are linearly independent.

v‾=aiv‾i\underline{\boldsymbol{ v}}=a_i\underline{\boldsymbol{ v}}_iv​=ai​v​i​, which we just used, is called a linear combination of the vectors v‾i\underline{\boldsymbol{ v}}_iv​i​. (Recall the Einstein summation convention)

Scalar product

The scalar product, u‾⋅v‾\underline{\boldsymbol{ u}}\cdot\underline{\boldsymbol{ v}}u​⋅v​, between two vectors, u‾\underline{\boldsymbol{ u}}u​ and v‾\underline{\boldsymbol{ v}}v​, is defined as

u‾⋅v‾=∣∣u‾∣∣∣∣v‾∣∣cos⁡(θ)\begin{aligned} \underline{\boldsymbol{ u}}\cdot\underline{\boldsymbol{ v}} = \left\vert\left\vert \underline{\boldsymbol{ u}}\right\vert\right\vert\left\vert\left\vert \underline{\boldsymbol{ v}}\right\vert\right\vert\cos(\theta) \end{aligned}u​⋅v​=∣∣u​∣∣∣∣v​∣∣cos(θ)​

where ∣∣u‾∣∣\left\vert\left\vert \underline{\boldsymbol{ u}}\right\vert\right\vert∣∣u​∣∣ is the length of the vector u‾\underline{\boldsymbol{ u}}u​, and θ\thetaθ is the angle between vectors u‾\underline{\boldsymbol{ u}}u​ and v‾\underline{\boldsymbol{ v}}v​. Note that we do not require a coordinate system to define this operation!

From this definition, it follows that ∣∣u‾∣∣=u‾⋅u‾\left\vert\left\vert \underline{\boldsymbol{ u}}\right\vert\right\vert = \sqrt{\underline{\boldsymbol{ u}}\cdot\underline{\boldsymbol{ u}}}∣∣u​∣∣=u​⋅u​​ as in this case, θ=0\theta=0θ=0. Hence, u‾⋅u‾=0⇔u‾=0‾\underline{\boldsymbol{ u}}\cdot\underline{\boldsymbol{ u}}=0 \Leftrightarrow \underline{\boldsymbol{ u}}=\underline{\boldsymbol{ 0}}u​⋅u​=0⇔u​=0​. Furthermore, if u‾\underline{\boldsymbol{ u}}u​ and v‾\underline{\boldsymbol{ v}}v​ are perpendicular vectors, then θ=π/2\theta=\pi/2θ=π/2 and thus u‾⋅v‾=0\underline{\boldsymbol{ u}}\cdot\underline{\boldsymbol{ v}}=0u​⋅v​=0.

Cross product

The cross (or vector) product, u‾×v‾\underline{\boldsymbol{ u}}\times\underline{\boldsymbol{ v}}u​×v​, between two vectors, u‾\underline{\boldsymbol{ u}}u​ and v‾\underline{\boldsymbol{ v}}v​, is defined as

u‾×v‾=∣∣u‾∣∣∣∣v‾∣∣sin⁡(θ)n‾\begin{aligned} \underline{\boldsymbol{ u}}\times\underline{\boldsymbol{ v}} = \left\vert\left\vert \underline{\boldsymbol{ u}}\right\vert\right\vert\left\vert\left\vert \underline{\boldsymbol{ v}}\right\vert\right\vert\sin(\theta) \underline{\boldsymbol{ n}} \end{aligned}u​×v​=∣∣u​∣∣∣∣v​∣∣sin(θ)n​​

Basis system

Above, we have seen that many properties of vectors can be defined without a basis system, simply by considering a vector as an object with a length and a direction. For these definitions, there is no need to introduce a coordinate system. However, if we want to perform actual calculations, everyting becomes easier if we define a base system from which we measure all quantities. Using a set of predefined so-called base vectors, we can express other vector quantities by using a linear combination of these base vectors. To ensure that this description is unique, we require that the base vectors are linearly independent. Otherwise, we can have the same vector described by different linear combinations.

Let's consider a basis system described by the linear independent vectors e‾i\underline{\boldsymbol{ e}}_ie​i​. A vector v‾\underline{\boldsymbol{ v}}v​, can then be described by the unique coefficients viv_ivi​ (considering the e‾i\underline{\boldsymbol{ e}}_ie​i​ basis system) by v‾=vie‾i\underline{\boldsymbol{ v}}=v_i\underline{\boldsymbol{ e}}_iv​=vi​e​i​. If we take the scalar product u‾⋅v‾\underline{\boldsymbol{ u}}\cdot\underline{\boldsymbol{ v}}u​⋅v​, we get

u‾⋅v‾=uivje‾i⋅e‾j\begin{aligned} \underline{\boldsymbol{ u}}\cdot\underline{\boldsymbol{ v}} = u_i v_j \underline{\boldsymbol{ e}}_i \cdot \underline{\boldsymbol{ e}}_j \end{aligned}u​⋅v​=ui​vj​e​i​⋅e​j​​

resulting in the metric coefficients eij=e‾i⋅e‾je_{ij}=\underline{\boldsymbol{ e}}_i \cdot \underline{\boldsymbol{ e}}_jeij​=e​i​⋅e​j​ for the coordinate system. This coefficient complicates our calculations, but by choosing our basis vectors cleverly, we can make it become δij\delta_{ij}δij​, resulting in u‾⋅v‾=uivi\underline{\boldsymbol{ u}}\cdot\underline{\boldsymbol{ v}} = u_i v_iu​⋅v​=ui​vi​. But how can we choose e‾i\underline{\boldsymbol{ e}}_ie​i​, such that e‾i⋅e‾j=δij\underline{\boldsymbol{ e}}_i \cdot \underline{\boldsymbol{ e}}_j=\delta_{ij}e​i​⋅e​j​=δij​?

The solution is an orthonormal coordinate system (Cartesian coordinate system). The ortho comes from orthogonal, implying perpendicularity between the base vectors. If we consider the scalar product, this implies that e‾i⋅e‾j=0\underline{\boldsymbol{ e}}_i\cdot\underline{\boldsymbol{ e}}_j=0e​i​⋅e​j​=0 if i≠ji\neq ji=j. The normal comes from that the base vectors are normalized, i.e. have unit length (∣∣e‾i∣∣=1\left\vert\left\vert \underline{\boldsymbol{ e}}_i\right\vert\right\vert=1∣∣e​i​∣∣=1). This implies that e‾i⋅e‾j=1\underline{\boldsymbol{ e}}_i\cdot\underline{\boldsymbol{ e}}_j=1e​i​⋅e​j​=1 if i=ji=ji=j. So with these two choices, we have exactly the definition of δij\delta_{ij}δij​, for our orthonormal coordinate system with basis vectors e‾i\underline{\boldsymbol{ e}}_ie​i​:

e‾i⋅e‾j=δij={1i=j0i≠j\begin{aligned} \underline{\boldsymbol{ e}}_i\cdot\underline{\boldsymbol{ e}}_j = \delta_{ij} = \left\lbrace\begin{matrix} 1 & i=j \\ 0 & i\neq j\end{matrix}\right. \end{aligned}e​i​⋅e​j​=δij​={10​i=ji=j​​

Scalar product in orthonormal basis system

As mentioned above, the scalar product between two vectors, u‾=uie‾i\underline{\boldsymbol{ u}}=u_i\underline{\boldsymbol{ e}}_iu​=ui​e​i​ and v‾=vie‾i\underline{\boldsymbol{ v}}=v_i\underline{\boldsymbol{ e}}_iv​=vi​e​i​, in an orthonormal coordinate system, e‾i\underline{\boldsymbol{ e}}_ie​i​, becomes

u‾⋅v‾=uivi\begin{aligned} \underline{\boldsymbol{ u}}\cdot\underline{\boldsymbol{ v}} = u_i v_i \end{aligned}u​⋅v​=ui​vi​​

Cross product in orthonormal basis system

As seen from the definition, the cross product is more complicated. Again, we consider two vectors, u‾=uie‾i\underline{\boldsymbol{ u}}=u_i\underline{\boldsymbol{ e}}_iu​=ui​e​i​ and v‾=vie‾i\underline{\boldsymbol{ v}}=v_i\underline{\boldsymbol{ e}}_iv​=vi​e​i​, in an orthonormal coordinate system, e‾i\underline{\boldsymbol{ e}}_ie​i​. The cross-product then becomes

u‾×v‾=(uie‾i)×(vje‾j)=uivje‾i×e‾j\begin{aligned} \underline{\boldsymbol{ u}}\times\underline{\boldsymbol{ v}} = (u_i\underline{\boldsymbol{ e}}_i) \times (v_j\underline{\boldsymbol{ e}}_j) = u_i v_j \underline{\boldsymbol{ e}}_i\times\underline{\boldsymbol{ e}}_j \end{aligned}u​×v​=(ui​e​i​)×(vj​e​j​)=ui​vj​e​i​×e​j​​

Based on the definition of the cross product and the orthonormal coordinate system, we know the following in 3 dimensions

e‾i×e‾j={±e‾ki≠jk≠i,j0‾i=j\begin{aligned} \underline{\boldsymbol{ e}}_i\times\underline{\boldsymbol{ e}}_j = \left\lbrace\begin{matrix} \pm\underline{\boldsymbol{ e}}_k & i\neq j & k\neq i,j\\ \underline{\boldsymbol{ 0}} & i=j & \end{matrix}\right. \end{aligned}e​i​×e​j​={±e​k​0​​i=ji=j​k=i,j​​

Hence, the direction in the case that i≠ji\neq ji=j is not properly defined. This depends on the order of our basis vectors. We therefore need to introduce this order in our definition of our coordinate system and we will choose a right-handed coordinate system (see the illustration used to define the cross product). For a coordinate system, this implies that e‾3=e‾1×e‾2\underline{\boldsymbol{ e}}_3=\underline{\boldsymbol{ e}}_1\times\underline{\boldsymbol{ e}}_2e​3​=e​1​×e​2​. Using the Levi-Civita symbol, εijk\varepsilon_{ijk}εijk​, we then have

e‾i×e‾j=εijke‾k\begin{aligned} \underline{\boldsymbol{ e}}_i\times\underline{\boldsymbol{ e}}_j = \varepsilon_{ijk} \underline{\boldsymbol{ e}}_{k} \end{aligned}e​i​×e​j​=εijk​e​k​​

in a right-handed orthonormal coordinate system. The cross product for two general vectors is then

u‾×v‾=uivjεijke‾k\begin{aligned} \underline{\boldsymbol{ u}}\times\underline{\boldsymbol{ v}} = u_i v_j \varepsilon_{ijk} \underline{\boldsymbol{ e}}_k \end{aligned}u​×v​=ui​vj​εijk​e​k​​

Unless otherwise specified, coordinate systems discussed on this webpage are right-handed and orthonormal.
